# Installing CodeLookup

Install CodeLookup for all supported agents on your machine in one command.

## One-Liner Install

### macOS / Linux
```bash
curl -fsSL https://raw.githubusercontent.com/username/CodeLookup/main/install.sh | bash
```

### Windows (PowerShell)
```powershell
irm https://raw.githubusercontent.com/username/CodeLookup/main/install.ps1 | iex
```

## Local Installation (Cloned Repo)

```bash
git clone https://github.com/username/CodeLookup.git
cd CodeLookup
node bin/install.js
```
