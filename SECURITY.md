# Security & Privacy Policy

## Reporting Vulnerabilities

If you find a security vulnerability, please report it by opening an issue or contacting the repository owner directly.

## Privacy Statement

CodeLookup runs entirely on your local machine:
- **No Telemetry**: No tracking, metrics, or logs are uploaded.
- **No Remote Calls**: The scripts do not make external HTTP requests.
- **Local Isolation**: The dependency graph data is written to a local `.codelookup/` directory inside your workspace and is ignored by default in Git.
